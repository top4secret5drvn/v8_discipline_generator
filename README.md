# v8_discipline_generator
Discipline report generator with habit tracking, streaks, attribute system (I,S,W,E,C,H,ST,$). SQLite storage, daily stats, previous day comparison. Built with Flask &amp; vanilla JS. / Генератор отчётов дисциплины: трекинг привычек, стрики, характеристики (I,S,W,E,C,H,ST,$). Хранение в SQLite, дневная статистика. Стек: Flask + чистый JS.
